# QR Code Studio 🚀

## 📝 About the Site
A professional SaaS-style web application that converts any link, text, or URL into a clean, scannable QR code instantly. The site features a modern glassmorphic interface, a smooth 3-step integrated user workflow, an instant one-click download feature, and an ambient video background.

---

## ⚙️ Working (How it Works)

### Using the Application:
1. **Copy Link:** Copy any target website URL or plain text to your clipboard.
2. **Paste & Generate:** Paste the text into the input field and click the **Generate QR Code** button.
3. **Save Graphic:** The custom high-resolution QR code appears instantly. Scan it directly from the screen or click the **Download QR Code** button to save it as a `.png` file.

### Running it Locally:
* Open this project directory inside **VS Code**.
* Right-click on `index.html` and select **Open with Live Server**.
